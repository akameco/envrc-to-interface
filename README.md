> your .envrc to TypeScript interface


## Install

```
$ npm install envrc-to-interface
$ yarn add envrc-to-interface
```


## Usage

```js
const envrcToInterface = require('envrc-to-interface');

envrcToInterface('unicorns');
//=> 'unicorns & rainbows'
```
